TP 0 : Système de Gestion d'Inventaire (Jakarta EE & Hibernate)
Ce projet est une application web complète développée dans le cadre du cours Développement JakartaEE : Spring. Il met en œuvre les technologies standards de Java Enterprise Edition pour la gestion d'un inventaire.
🏗️ Architecture et Design Patterns
L'application repose sur une architecture MVC (Modèle-Vue-Contrôleur) structurée comme suit :
Modèle : Gestion de la persistance avec Hibernate et un DAO générique.
Vue : Interfaces utilisateur dynamiques réalisées en JSP.
Contrôleur : Logique métier pilotée par des Servlets.
🛠️ Stack Technique
Framework : Jakarta EE (JSP, Servlet).
ORM : Hibernate avec écouteur de session (Listener).
Build Tool : Maven.
Serveur : Apache Tomcat.
